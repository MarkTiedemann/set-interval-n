
# set-interval-n

**setInterval() _n_ times.**

## Installation

```
npm install set-interval-n
```

## Usage

```js
const setIntervalN = require('set-interval-n')

setIntervalN(i => console.log(i), 1000, n = 3)

// ==> 0
    // ==> 1
        // ==> 2
```
